# Paper Abstract Clustering for Literature Review

## Task Description

You are provided with a CSV file named `abstracts.csv` located in the workspace directory. This file contains metadata about academic papers, with the following columns:

- `paper_id`: Unique identifier for each paper
- `title`: Title of the paper
- `abstract`: Abstract text of the paper
- `year`: Publication year

Your task is to perform a literature review clustering based on the abstracts:

1. **Read** the `abstracts.csv` file.
2. **Compute** the TF-IDF matrix of the paper abstracts to represent their textual content numerically.
3. **Cluster** the papers into 5 clusters using the K-Means algorithm (k=5) based on the TF-IDF features.
4. **Identify** the top terms that characterize each cluster.
5. **Write** the results to a JSON file named `topic_clusters.json` in the workspace directory.


## Output Format

The output JSON file `topic_clusters.json` should contain:

- `clusters`: A dictionary mapping each cluster ID (0 to 4) to a list of `paper_id`s assigned to that cluster.
- `top_terms`: A dictionary mapping each cluster ID to a list of the top 10 terms that best represent that cluster.
- `review_outline`: A list of 5 strings, each string is a short summary or title representing the cluster topic, generated by concatenating the top terms for that cluster.

Example structure:

```json
{
  "clusters": {
    "0": ["P001", "P005", ...],
    "1": ["P002", "P006", ...],
    ...
  },
  "top_terms": {
    "0": ["neural", "network", "deep", "learning", ...],
    "1": ["graph", "algorithm", "network", ...],
    ...
  },
  "review_outline": [
    "neural network deep learning ...",
    "graph algorithm network ...",
    ...
  ]
}
```

## Requirements

- Use TF-IDF vectorization on the abstracts.
- Use K-Means clustering with k=5.
- Extract the top 10 terms per cluster by averaging cluster centroid weights.
- Generate a simple review outline by joining the top terms per cluster.
- The solution should be robust and handle the provided dataset.


## Files

- Input: `workspace/abstracts.csv`
- Output: `workspace/topic_clusters.json`


## Evaluation

Your submission will be evaluated by:

- Correctness of clustering assignments.
- Correct extraction of top terms per cluster.
- Proper JSON output format.
- Code robustness and reproducibility.
