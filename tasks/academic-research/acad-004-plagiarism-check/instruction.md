# Document Similarity and Plagiarism Detection

## Objective

You are given a workspace containing a directory `workspace/documents/` with 5 or more text files. Your task is to analyze these documents to detect potential plagiarism by computing pairwise similarity scores between all documents.

## Requirements

1. **Read all text files** from the directory `workspace/documents/`.
2. **Compute pairwise similarity** between every pair of documents using two methods:
   - **3-gram Jaccard similarity:**
     - Extract all character 3-grams from each document.
     - Compute the Jaccard similarity between the sets of 3-grams for each pair.
   - **Cosine similarity on TF-IDF vectors:**
     - Compute TF-IDF vectors for each document.
     - Compute cosine similarity between each pair of TF-IDF vectors.
3. **Output two files** in the workspace root:
   - `similarity_matrix.csv`: A CSV file with columns:
     - `doc1`: filename of first document
     - `doc2`: filename of second document
     - `jaccard_3gram`: Jaccard similarity score (float between 0 and 1)
     - `cosine_tfidf`: Cosine similarity score (float between 0 and 1)

     Include all unique pairs (doc1 < doc2 lex order) without duplicates or self-pairs.

   - `plagiarism_report.json`: A JSON file listing all document pairs where **either** similarity measure is greater than 0.3. The format should be a list of objects:

     ```json
     [
       {
         "doc1": "fileA.txt",
         "doc2": "fileB.txt",
         "jaccard_3gram": 0.45,
         "cosine_tfidf": 0.52
       },
       ...
     ]
     ```

## Notes

- Use character 3-grams (sliding window of length 3) for Jaccard similarity.
- Use standard TF-IDF vectorization with default parameters.
- Similarity scores should be floats rounded to at least 4 decimal places.
- Only consider pairs where `doc1` filename is lexicographically less than `doc2` to avoid duplicates.
- The workspace directory structure and files are generated by the setup script.

## Deliverables

Your solution should read the input files, compute similarities as specified, and write the two output files in the workspace root:

- `similarity_matrix.csv`
- `plagiarism_report.json`

These files will be verified for correctness.

Good luck!
